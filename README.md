# Spring Boot + Vue3 前后端分离，实战wiki知识库系统

## 项目初始化
* 需要本地安装好idea, nodejs，jdk1.8, mysql8.0/5.7, navicat（数据库可视化工具）
* 将下载好的源码，用idea打开
* 刷新maven依赖
* 安装vue cli
* 初始化web模块
```
cd web
npm install
```
* 新建数据库，数据库配置在application.properties

## 项目启动
* 启动服务端：WikiApplication
* 启动前端网站：web\package.json

## 页面访问
* 网站地址: http://localhost:8080<br>
  初始用户名密码：test/test
